# Car-Rental-System
A console-based Car Rental System developed in C++ using Object-Oriented Programming (OOP) concepts. The system allows users to view available cars, rent vehicles, return them, and manage customer and booking records efficiently.
  Key Features
View available cars
Rent and return cars
Customer & booking management
Rental cost calculation
 OOP Concepts

Encapsulation, Inheritance, Polymorphism, and Abstraction
